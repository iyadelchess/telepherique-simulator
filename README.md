# Simulation d’un système de transport automatisé (Téléphérique)

## Description
Projet en cours de développement visant à simuler le fonctionnement d’un téléphérique à travers une approche orientée objet en C++.

Le projet cherche à reproduire le comportement d’un système réel en intégrant une logique de simulation dynamique proche du temps réel, avec plusieurs composants interagissant entre eux (stations, cabines, système de contrôle).

## Fonctionnalités actuelles
- Structure de base du projet (classes principales : Cabine, Station, gestionnaire)
- Début de modélisation du système
- Mise en place de la logique initiale de simulation

## En cours de développement
- Simulation en temps réel du déplacement des cabines
- Gestion des états du système (arrêt, déplacement, attente)
- Synchronisation des interactions entre composants
- Amélioration de l’architecture logicielle

## Objectifs
- Reproduire un système automatisé proche d’un système réel
- Implémenter une logique de contrôle-commande simplifiée
- Simuler un comportement dynamique en temps réel
- Approfondir les concepts liés aux systèmes embarqués

## Technologies utilisées
- C++
- Programmation orientée objet
- Git / GitHub
